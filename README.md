### Упаковка в Docker Compose
В данном проекте происходит упаковка приложения js-fastify-blog в Docker образ. Docker Compose используется для настройки окружения для разработки, запуска тестов и CI. После успешного выполнения CI собирается образ приложения и публикуется на hub.docker.com. Это сделает настройку и запуск приложения делом на несколько минут, без инструкций.
